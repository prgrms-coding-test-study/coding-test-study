## 💭 아이디어
- deque 기반 너비 우선 탐색 활용
- 두 번의 너비 우선 탐색 수행
    1) 시작 지점 -> 레버
    2) 레버 -> 출구

## 🧑‍💻 코드
<!-- 작성한 코드를 백틱으로 감싸 넣어주세요 --> 
```python
from collections import deque

INF = -1;

START = 'S';
EXIT = 'E';
LEVER = 'L';
WALL = 'X';

X = 0;
Y = 1;

### 상하좌우 이동
DIR = [[-1, 0], [1, 0], [0, -1], [0, 1]];

def solution(maps):   
    row = len(maps)
    col = len(maps[0])
    
    start = [];
    exit = [];
    lever = [];
    
    ### 미로 순회
    for i in range(row):
        for j in range(col):
            ### 특수 좌표 저장
            if maps[i][j] == START:
                start = [i, j];
            elif maps[i][j] == EXIT:
                exit = [i, j];
            elif maps[i][j] == LEVER:
                lever = [i, j];
    
    ### start -> lever 최소 이동 시간 계산
    first_result = move(maps, row, col, start, lever)
    
    if first_result == INF:
        return INF
    
    ### lever -> exit 최소 이동 시간 계산
    second_result = move(maps, row, col, lever, exit)
    
    if second_result == INF:
        return INF
    
    return first_result + second_result

def move(maps, row, col, src, dest) -> int:
    queue = deque()
    times = [[INF] * col for _ in range(row)]
    
    queue.append(src);
    times[src[X]][src[Y]] = 0;
    
    while queue:
        x, y = queue.popleft()
        
        if [x, y] == dest:
            return times[x][y]
        
        for dx, dy in DIR:
            nx = x + dx
            ny = y + dy
            
            if nx < 0 or nx >= row or ny < 0 or ny >= col:
                continue
            if times[nx][ny] != INF:
                continue
            if maps[nx][ny] == WALL:
                continue
            
            queue.append([nx, ny])
            times[nx][ny] = times[x][y] + 1
    
    return INF
```

## ⏰ 복잡도
- 시간 복잡도: O(R * C)
- 공간 복잡도: O(R * C)
